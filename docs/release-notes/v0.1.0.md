# Suwol Visual Reference v0.1.0

Initial public release.

## Downloads

- `SuwolVisualReference-0.1.0-win-x64.zip`
- `SuwolVisualReference-0.1.0-linux-x64.zip`
- `SuwolVisualReference-0.1.0-checksums.txt`

## Highlights

- Local visual reference libraries for game developers.
- File and recursive folder import.
- Thumbnails, previews, tags, memo, rating, favorites, collections, and smart folders.
- Duplicate resolution, trash, restore, and permanent delete.
- Desktop-style selection and multi-selection batch operations.
- Large image viewer.
- Korean and English UI.
- Codex Markdown export.
- Windows and Linux ZIP distribution.

## Known Issues

- Windows builds are not code-signed and may trigger SmartScreen warnings.
- macOS builds are not distributed yet.
- Linux is distributed as a ZIP archive.
- Advanced video and audio preview workflows are not included yet.
- Automatic updates are not included.
