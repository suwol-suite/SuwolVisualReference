# Suwol Visual Reference v0.2.0 Draft

Unreleased planning notes. The application version remains `0.1.2` until an explicit version bump and release task is approved.

## Draft Highlights

- Added dense list view for asset discovery.
- Added saved list column visibility, order, and width controls.
- Added structured sort controls and DB-backed order clauses.
- Added advanced asset filters for practical large-library narrowing.
- Added collection manager with rename, description, color, cover, and delete actions.
- Added collection cover fallback thumbnails.
- Added manual collection ordering with drag handles and move buttons.
- Added smart folder manager with multi-condition editing and result count previews.
- Added smart folder UI fields for media type, width, height, tag exclusion, memo, and source URL rules.
- Added safer media import handling for GIF, SVG, and video records. GIFs get first-frame thumbnails and viewer playback from the original file. SVGs use raster previews. Video thumbnails are optional external-ffmpeg-only and fall back to placeholders with warnings.
- Added GIF play/pause controls in the viewer while keeping grid/list GIF thumbnails static.
- Added palette color search using RGB palette channels, swatch-to-filter interaction, color tolerance, minimum color-share controls, and removable active filter chips.
- Added custom Codex Markdown export templates with built-in read-only templates, duplicate/create/edit/delete custom templates, section reorder controls, placeholder insertion, preview, and unknown placeholder warnings.
- Added Help/About content for keyboard shortcuts, repository, license, third-party notices, known issues, and optional FFmpeg policy.
- Expanded smoke coverage for media fallback import, color filtering, export templates, asset organization, collection ordering, and smart folder workflows.

## Release Status

No tag, GitHub Release, or push is part of this draft. Run the release checklist only after the version bump is explicitly requested.
