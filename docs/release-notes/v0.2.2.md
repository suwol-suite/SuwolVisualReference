# Suwol Visual Reference v0.2.2

This patch release adds the release automation foundation for later macOS arm64 distribution while keeping the normal Windows/Linux release path unchanged.

## Added

- Reusable Electron release standard documentation.
- Manual macOS arm64 build diagnostics workflow for a trusted self-hosted Apple Silicon runner.
- Manual workflow for attaching signed and notarized macOS arm64 assets to an existing GitHub Release.
- macOS hardened runtime entitlements.
- Helper scripts for icon checks, DMG notarization, and native payload re-signing.

## Changed

- Checksum generation and verification now include `.dmg` artifacts and `latest-mac.yml` when macOS release assets are attached.
- Packaging and release documentation now describe the Windows/Linux core release first, then optional macOS attachment.

## Notes

- Windows and Linux release artifacts remain the core public release assets.
- macOS artifacts are still produced only through the separate manual diagnostics and attachment flow.
- No universal or Intel macOS build is included.
